## 📖 Overview

Complete test automation framework for hotel booking websites using **Selenium WebDriver** and **TestNG**. This framework automates the entire hotel booking workflow including search, filtering, hotel selection, booking, payment, and confirmation.

## ✨ Features

- ✅ **Hotel Search** - Search hotels by city, check-in, and check-out dates
- ✅ **Filter Results** - Filter hotels by price, rating, and amenities
- ✅ **Sort Results** - Sort hotels by price, rating, or popularity
- ✅ **Hotel Details** - View hotel name, address, description, price, and availability
- ✅ **Complete Booking Flow** - End-to-end booking process automation
- ✅ **Payment Processing** - Credit card payment validation
- ✅ **Booking Confirmation** - Verify booking success and retrieve booking number
- ✅ **Cross-Browser Testing** - Chrome, Firefox, and Edge support
- ✅ **Parallel Execution** - Run tests in parallel for faster execution
- ✅ **Extent Reports** - Detailed HTML reports with screenshots
- ✅ **Data-Driven Testing** - Test with multiple data sets from Excel/JSON
- ✅ **Retry Mechanism** - Auto-retry failed tests (2 retries)
- ✅ **Screenshots** - Automatic screenshots on test failure
- ✅ **Logging** - Detailed logs using Log4j
